# Iconize v0.1

## This plugin may or may not do this:
- fix icon name (to lowercase)
- fix vector name (to 'ic')
- outline strokes
- flatten vectors
- set same color style for all icons (more orange in your life)
- set width and hight to 16px
- set constraints for vector to 'SCALE'

## Usage:
- select icon, group of icons or frame containing icons
- apply

## Recommended shortcut:
**⇧⌃⌘I**
